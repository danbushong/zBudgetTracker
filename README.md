# zBudgetTracker
  ## Description
  Creates a budget tracker that can operate online and offline.
  ## Table of Contents
  * [Installation](#Installation)
  * [Usage](#Usage)

  ## Installation
  npm install
  <br>
  npm start

  Keep track of your finances, and see them displayed with a chart.
